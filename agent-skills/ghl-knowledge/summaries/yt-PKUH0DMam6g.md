# GoHighLevel App Marketplace - How to Install Apps in GoHighLevel (Tutorial)

**Source:** https://www.youtube.com/watch?v=PKUH0DMam6g
**Author:** Tin Rovic

Here are the key takeaways from the GoHighLevel tutorial video:

**Key Features:**
* GoHighLevel App Marketplace allows users to install apps to extend the platform's functionality
* Two marketplaces: one for GoHighLevel services and another for app marketplace
* Apps can be free or paid, with varying levels of access and functionality
* Ability to install apps on sub-accounts

**Step-by-Step Setup Instructions:**
* Switch to Agency View
* Navigate to the App Marketplace
* Browse available apps and click on the desired app to view its details
* Click "Install" to install the app on a sub-account
* Log in to the app's website to complete the installation process

**API/Automation Capabilities:**
* No explicit API or automation capabilities mentioned in the video
* However, the ability to install apps on sub-accounts suggests potential for automation and integration with other tools

**Potential Marketplace App Ideas:**
* Learning platforms (e.g. AI2)
* Productivity and time-saving tools
* Integration with other marketing or business apps (e.g. Shopify)
* Custom workflow automation tools
* Data analysis and reporting tools
* Customer relationship management (CRM) tools

**Additional Notes:**
* The video highlights the potential time-saving benefits of using apps in GoHighLevel
* The app installation process is compared to installing apps on Shopify, suggesting a similar level of ease and flexibility.

# Build A GoHighLevel Marketplace App in 20 Minutes Or Less! (Updated 2026)

**Source:** https://www.youtube.com/watch?v=wmoshlenptM
**Author:** BetterBots

Here are the key features, step-by-step setup instructions, API/automation capabilities, and potential marketplace app ideas extracted from the tutorial video:

**Key Features:**

* GoHighLevel Marketplace App setup
* OOTH 2 (OAuth 2.0) setup for authentication
* Make.com (or similar no-code automation tool) integration for server-side processing
* GoHighLevel Website Builder for creating a landing page
* Custom script element for adding code to the landing page
* Redirect URL setup for whitelabeled apps

**Step-by-Step Setup Instructions:**

* Create a developer account on marketplace.go.gohighlevel.com/signup
* Create a new app and choose "Public" and "White Label" options
* Upload a logo and choose a category for the app
* Set up advanced settings:
	+ Choose scopes (e.g. contacts read and contacts)
	+ Add a redirect URL
* Create a key pair and copy the client ID and secret
* Set up a server instance using Make.com (or similar tool)
* Create a web hook and copy the URL
* Build a landing page using GoHighLevel Website Builder
* Add a custom script element to the landing page
* Copy and paste code into the script element
* Replace the placeholder URL with the web hook URL
* Save and publish the page
* Connect a domain to the site (if using a whitelabeled app)

**API/Automation Capabilities:**

* OOTH 2 (OAuth 2.0) for authentication
* Make.com (or similar tool) for server-side processing
* Web hooks for triggering automation scenarios
* API integration for exchanging code for tokens
* Lookup step for verifying token permissions
* Saving tokens in a spreadsheet for future use

**Potential Marketplace App Ideas:**

* Integrating third-party services with GoHighLevel (e.g. CRM, email marketing, etc.)
* Creating custom automation scenarios for GoHighLevel users
* Building white-labeled apps for clients or partners
* Developing custom landing pages for GoHighLevel users
* Creating API integrations for other services (e.g. payment gateways, etc.)

Note: The tutorial video is focused on building a specific type of app (OOTH 2) and may not cover all possible features and capabilities of the GoHighLevel platform.

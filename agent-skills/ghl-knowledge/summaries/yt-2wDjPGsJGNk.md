# Build A GoHighLevel App in 30 minutes!

**Source:** https://www.youtube.com/watch?v=2wDjPGsJGNk
**Author:** BetterBots

Here are the key features, step-by-step setup instructions, API/automation capabilities, and potential marketplace app ideas extracted from the tutorial video:

**Key Features:**

* GoHighLevel Developer API and Portal
* Ability to create custom apps without writing code
* Scopes for different resources (e.g. conversations, contacts, opportunities)
* Webhook URL for receiving notifications
* Integromat integration for automating workflows
* OAuth redirect URLs for connecting sub-accounts to apps

**Step-by-Step Setup Instructions:**

* Sign up for a developer account on marketplace.gohighlevel.com
* Create a new app and add scopes for the resources you want to access
* Set up a webhook URL to receive notifications
* Create an Integromat scenario to automate workflows
* Set up OAuth redirect URLs to connect sub-accounts to your app
* Use a funnel to handle the OAuth redirect process
* Add custom code to the funnel to redirect to the GoHighLevel OAuth marketplace URL

**API/Automation Capabilities:**

* Webhook URL for receiving notifications on conversation messages, contacts, and opportunities
* Ability to send messages as a user without being logged in using app credentials
* Integromat integration for automating workflows
* OAuth redirect URLs for connecting sub-accounts to apps
* API requests for creating, reading, updating, and deleting resources (e.g. conversations, contacts, opportunities)

* Integration app for integrating GoHighLevel with other third-party services (e.g. email marketing, payment gateways)

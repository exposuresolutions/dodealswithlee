# How to Use GoHighLevel AI Voice Agents: Ultimate 2025 Tutorial

**Source:** https://www.youtube.com/watch?v=VvYCVfoYuMk
**Author:** GHL Wizard

Here are the key points extracted from the tutorial video:

**Key Features:**
* AI voice agents can be trained to book clients 24/7 and deliver stellar customer service
* Ability to use any phone number, even outside of GoHighLevel, for AI voice agents
* Knowledge base is where the AI agent pulls data from to talk to leads
* Ability to upload files (coming soon) to train the AI agent
* Web crawler feature to train the AI agent on a website
* Multiple voices and languages available for the AI agent

**Step-by-Step Setup Instructions:**
* Start by creating a knowledge base in GoHighLevel
* Add the client's website to the knowledge base to train the AI agent on pricing, FAQs, etc.
* Use a prompt (provided in the video) to fast track the training process and add niche-specific data
* Copy and paste the prompt into a chat tool (e.g. ChatGBT) and plug in the niche (e.g. plumber)
* Copy the output and paste it into a Google sheet or webpage to create an extra database
* Add the link to the knowledge base to train the AI agent on the niche
* Create an AI voice agent and name it
* Choose a language and voice for the AI agent
* Set the time zone for the bot's inbound call
* Set the agent's initial message

**API/Automation Capabilities:**
* Ability to use webhooks to automate tasks (not explicitly mentioned in the video, but possible with GoHighLevel's API)
* Potential to integrate with other tools and services using GoHighLevel's API

**Potential Marketplace App Ideas:**
* AI voice agent templates for specific industries (e.g. plumbing, healthcare)
* Pre-built knowledge bases for common niches
* Integration with popular chat tools (e.g. ChatGBT) for easy training of AI agents
* Custom voice and language packs for AI agents
* Automated workflows for AI voice agents using GoHighLevel's API

**Additional Notes:**
* The importance of properly training the AI agent on a knowledge base and niche-specific data
* The potential for uploading files (e.g. voice files) to train the AI agent in the future
* The ability to edit and delete knowledge bases and AI voice agents in GoHighLevel

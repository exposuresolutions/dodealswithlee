# How to Use GoHighLevel AI Voice Agent (Full 2025 Tutorial)

**Source:** https://www.youtube.com/watch?v=TuAc-z5Dw5k
**Author:** ItsKeaton

Here are the key features, setup steps, API/automation capabilities, and potential marketplace app ideas extracted from the GoHighLevel tutorial video:

**Key Features:**

* AI Voice Agent that answers calls, qualifies leads, and books appointments 24/7
* Customizable agent name, language, and voice
* Knowledge base creation to train the personality of the bot
* Ability to collect specific details from callers (e.g. name, email)
* Option to trigger a workflow when a call is completed
* Phone number masking and availability settings
* Testing feature to ensure the agent is working correctly

**Step-by-Step Setup Instructions:**

1. Create a new AI agent:
	* Give the agent a name
	* Write the business's name
	* Select the language and voice
	* Preview the voice
	* Select the time zone
2. Define the agent's goals:
	* Create a knowledge base (if not already created)
	* Add sources to the knowledge base (e.g. website URL)
	* Select the knowledge base for the agent
	* Choose the details to collect from callers (e.g. name, email)
	* Option to trigger a workflow when a call is completed
3. Set up phone and availability:
	* Buy a phone number (if not already purchased)
	* Connect the phone number to the AI agent
	* Enable the AI agent as a backup to the phone number

**API/Automation Capabilities:**

* Ability to trigger a workflow when a call is completed
* Option to automate tasks after a call is completed (e.g. sending follow-up emails)
* Potential for integrating with other GoHighLevel features (e.g. email marketing, CRM)

**Potential Marketplace App Ideas:**

* Pre-built knowledge bases for specific industries (e.g. healthcare, finance)
* Customizable AI agent templates for different business types (e.g. e-commerce, services)
* Integration with other GoHighLevel features (e.g. email marketing, CRM) to create a comprehensive automation solution
* AI-powered chatbot solutions for websites and social media platforms
* Industry-specific AI agent solutions (e.g. medical appointment scheduling, financial consulting)

## REBILLING SETUP IN GHL

### For DDWL (on $497 plan — SaaS Mode available)

1. **Agency Settings > Reselling** — Enable AI Employee reselling
2. **Per sub-account** — Set rebilling multiplier (e.g., 3x–5x markup)
3. **SaaS Configurator** — Set monthly subscription price per sub-account
4. **Client pays via Stripe** — Connected to DDWL's Stripe account
5. **GHL auto-bills** — Usage tracked and billed automatically

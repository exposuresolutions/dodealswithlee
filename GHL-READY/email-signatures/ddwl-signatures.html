📧 How to Install

Gmail:

  1. Click "Copy Signature" for the person below
  2. Open Gmail → Settings (gear icon) → See all settings
  3. Scroll to "Signature" section → Click "+ Create new"
  4. Name it (e.g. "DDWL Signature") → Paste (Ctrl+V / Cmd+V)
  5. Set as default for new emails & replies → Save Changes

Outlook (Desktop):

  1. Click "Copy Signature" below
  2. Open Outlook → File → Options → Mail → Signatures
  3. Click New → Name it "DDWL" → Paste in the editor
  4. Set as default for new messages & replies → OK

Outlook (Web / Microsoft 365):

  1. Click "Copy Signature" below
  2. Open Outlook web → Settings (gear) → View all Outlook settings
  3. Mail → Compose and reply → Paste in the signature box
  4. Set as default → Save
